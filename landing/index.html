API Reference

All you need — three endpoints.

POST /v1/parse/invoice

Upload a PDF or image file. Returns structured invoice data as JSON. Max file size: 10MB. Requires authentication.

GET /v1/usage

Check your current plan, calls used, calls remaining, and daily/monthly breakdown. Requires authentication.

POST /v1/register

Register for a free API key. Send JSON body: {"email": "you@example.com"}. Returns your API key (store it — shown once).

Python 
import requests

response = requests.post(
    "https://invoiceparser.dev/v1/parse/invoice",
    headers={"X-API-Key": "inv_your_key_here"},
    files={"file": open("invoice.pdf", "rb")}
)

invoice = response.json()["data"]
print(f"Total: {invoice['currency']} {invoice['total']}")
